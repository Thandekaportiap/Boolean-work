<h1>Boolean-Work</h1>

## Run Locally
Clone the project
```bash
  https://github.com/Thandekaportiap/Boolean-work.git
```
Go to the project directory
```bash
  cd Boolean-work
```
Install dependencies
```bash
  npm install
```
Start the react server
```bash
  npm run dev
```
## Tech Stack
**Client:** React JS, HTML5, JavaScript, CSS3
